Disable Microsoft Windows Media Center Telemetry

ABOUT
Remove Telemetry reporting for the Microsoft Windows Media Center Program
Program (MS WMC)

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSWMC (Boolean) - Enable or disable MS WMC
		Y	- Enable (DEFAULT).
		N	- Disable.


NOTE
Disabling the plugin does not undo changes made in previous runs.
